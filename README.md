# BraniacLMS

Современная платформа продажи одежды. Прогрессивный взгляд на простые вещи.

Учебный проект интернет магазин одежды, который разработан при прохождений обучения.

## Стек

- Python > 3.10
  - isort, black, autoflake
  - Django < 3.3
  - Celery[Redis]
- VSCode
- SQLite 3
